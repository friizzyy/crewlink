# CrewLink Copy & Voice Guidelines

This document defines the tone and voice rules for all copy throughout the CrewLink platform.

## Brand Voice Pillars

### 1. Premium but Approachable
- Sound confident and polished, never desperate or salesy
- Be helpful without being patronizing
- Convey expertise without jargon

### 2. Calm and Intentional
- Every word should earn its place
- Prefer shorter, clearer sentences
- Avoid filler words and unnecessary qualifiers

### 3. Human and Authentic
- Sound like a knowledgeable friend, not a robot
- Use contractions naturally (it's, you're, we'll)
- Acknowledge the reader as capable adults

## Do's and Don'ts

### DO ✓

| Preferred | Example |
|-----------|---------|
| Clear, direct language | "Post a job in minutes" |
| Active voice | "We connect you with verified workers" |
| Specific benefits | "Background-checked professionals" |
| Natural contractions | "You'll get matched within hours" |
| Confident assertions | "Find the right help, fast" |

### DON'T ✗

| Avoid | Instead |
|-------|---------|
| "AI-powered" / "leveraging AI" | Describe the benefit directly |
| "Utilize" / "leverage" | "Use" |
| "Solutions" (generic) | Specific service name |
| "Revolutionize" / "Disrupt" | Concrete improvements |
| "Seamless" / "Frictionless" | How it's actually easy |
| "Robust" / "Scalable" | What it actually does |
| "Best-in-class" | Specific differentiator |
| "Synergy" / "Ecosystem" | Plain English |
| Multiple exclamation marks | One per page max |
| ALL CAPS for emphasis | Bold or context |

## AI-ish Phrases to Avoid

These phrases signal generic AI-generated content:

```
- "In today's fast-paced world..."
- "Whether you're looking to..."
- "We understand that..."
- "At CrewLink, we believe..."
- "Our state-of-the-art..."
- "Rest assured..."
- "With our innovative..."
- "Your satisfaction is our..."
- "We're committed to..."
- "Take your X to the next level"
- "Unlock your potential"
- "Empowering you to..."
- "Journey" (unless literal travel)
- "Elevate" (unless literal)
- "Curated" (unless truly hand-picked)
```

## Microcopy Guidelines

### Buttons & CTAs
- Use action verbs: "Post Job", "Find Work", "View Details"
- Be specific: "Send Message" not "Submit"
- Max 2-3 words when possible

### Form Labels
- Clear and descriptive: "Job Title", "Your Budget"
- Use sentence case, not Title Case
- Avoid ending with colons

### Error Messages
- Tell users what happened AND what to do
- Don't blame the user
- ✓ "Couldn't save changes. Check your connection and try again."
- ✗ "Error: Invalid input"

### Empty States
- Acknowledge the state
- Guide to next action
- ✓ "No messages yet. Start a conversation when you book a job."
- ✗ "Nothing to show."

### Loading States
- Be specific when possible
- ✓ "Finding jobs near you..."
- ✗ "Loading..."

### Success Messages
- Confirm what happened
- Suggest next steps when relevant
- ✓ "Job posted! Workers will start bidding shortly."
- ✗ "Success!"

## Capitalization

### Title Case
- Page titles
- Navigation items
- Button text (primary CTAs)

### Sentence Case
- Body copy
- Form labels
- Helper text
- Secondary buttons
- Toast messages

### Never ALL CAPS
- Except for acronyms (GPS, ID)
- Never for emphasis

## Numbers & Formatting

| Type | Format |
|------|--------|
| Money | $25, $1,500 (no cents unless relevant) |
| Time | 2pm, 3:30pm (no space, lowercase) |
| Dates | Jan 15, 2024 (abbreviated month) |
| Distance | 2.5 mi, 0.3 mi |
| Ratings | 4.8 (one decimal) |

## Pronouns

- Address users as "you" / "your"
- Refer to CrewLink as "we" / "our"
- Avoid "one" or "the user"
- Third parties: use their name or role ("Sarah", "your worker")

## Tone by Context

### Marketing Pages
- Confident, aspirational
- Focus on benefits
- Slightly more formal

### App Interface
- Helpful, efficient
- Focus on actions
- Casual but professional

### Errors & Warnings
- Direct, calm
- Focus on resolution
- Never alarming

### Notifications
- Brief, informative
- Action-oriented
- Friendly

## Common Phrases Dictionary

| Instead of | Write |
|------------|-------|
| "Click here" | "View jobs" (specific action) |
| "Submit" | "Post", "Send", "Save" |
| "Manage your account" | "Account settings" |
| "Get started today" | "Get started" |
| "Learn more" | "See how it works" |
| "N/A" | "—" or omit |
| "TBD" | Never ship this |
| "Coming soon" | Specific timeline or omit |

## Review Checklist

Before shipping copy, verify:

- [ ] No AI-ish phrases or corporate jargon
- [ ] Active voice used throughout
- [ ] Capitalization is consistent
- [ ] Numbers follow formatting rules
- [ ] Error messages include resolution steps
- [ ] CTAs use specific action verbs
- [ ] No unnecessary words that can be removed
- [ ] Tone matches context (marketing vs app vs error)
- [ ] Pronouns are consistent
- [ ] No placeholder text shipped

## Examples

### Job Card
✓ **Good:**
```
Deep Clean - 3 Bedroom Home
$120 · Flexible schedule · 2.3 mi away
Posted 2 hours ago · 4 bids
```

✗ **Avoid:**
```
DEEP CLEAN - 3 BEDROOM HOME!!!
Budget: $120.00 | Schedule: Flexible | Distance: 2.3 miles
This job was posted 2 hours ago and has received 4 bids so far.
```

### Onboarding
✓ **Good:**
```
Welcome to CrewLink
Let's set up your profile so you can start finding jobs that match your skills.
```

✗ **Avoid:**
```
Welcome to CrewLink! We're so excited to have you here!
In today's fast-paced gig economy, it's more important than ever to have a complete profile. Let's get you set up so you can start your journey to success!
```

### Error
✓ **Good:**
```
Couldn't load messages
Your connection dropped. Pull down to refresh.
```

✗ **Avoid:**
```
Error!
An unexpected error occurred while trying to fetch your messages. Please try again later or contact support if the problem persists.
```
